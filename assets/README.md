# Assets Klasörü

Bu klasör, PT Track web sitesi için gerekli tüm statik dosyaları içerir.

## Klasör Yapısı

- **images/** - Görseller ve ikonlar
- **css/** - CSS stil dosyaları
- **js/** - JavaScript dosyaları
- **fonts/** - Özel yazı tipleri

## Kullanım

Web sitesi içinde bu klasördeki dosyalara erişmek için:

```html
<!-- Görsel örneği -->
<img src="assets/images/logo.png" alt="PT Track Logo">

<!-- CSS örneği -->
<link rel="stylesheet" href="assets/css/styles.css">

<!-- JavaScript örneği -->
<script src="assets/js/app.js"></script>
```
